# board-online

### Requirements
 Python 3.10

### Setup develop environment
```bash
# Create virtual environment
python3.10 -m venv .venv

# Activate virtual env
# E.g
source.venv/bin/Activate

# Install dependencies
pip install pip-tools
make sync-deps
```
